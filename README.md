# 🚔 AI CRIME PATH RECONSTRUCTOR
### 🧠 Forensic Route Intelligence System

> **"Reconstruct Crime. Visualize Evidence. Solve Faster."**

---

## 📌 Overview
AI Crime Path Reconstructor is an intelligent forensic system designed to help investigators reconstruct crime routes, analyze evidence locations, and track suspect movements using interactive maps and AI-driven insights.

This project combines **AI + GIS + Data Visualization** to modernize crime investigation.

---

## 🎯 Key Features
- 🗺️ Route Reconstruction (Start → End mapping)
- 📍 Evidence Logging
- 🧠 AI-Based Analysis
- 👤 Suspect Tracking
- 🔥 Heatmap Visualization
- 📊 Distance & Speed Analysis
- 🧭 Zone Mapping & Measurement
- ⏱️ Crime Timeline Generation

---

## ⚙️ How It Works
1. 📍 Set Start & End Locations  
2. 🛣️ Generate Optimal Route  
3. 🧾 Log Evidence Points  
4. 📊 Analyze Movement Patterns  
5. 📈 Generate Insights & Reports
